# Random Password Generator

**Автор:** Хайруллин Айгиз

## Описание

Приложение с графическим интерфейсом для генерации случайных паролей с настраиваемыми параметрами:
- Длина пароля (от 4 до 32 символов)
- Использование цифр, букв и спецсимволов
- Сохранение истории генерации в JSON-файл
- Загрузка истории при запуске

## Установка и запуск

1. Убедитесь, что установлен Python 3.6+
2. Скачайте файл `main.py`
3. Запустите:
```bash
python main.py
